2013 yılından beri farklı projeler ve farklı şirketler için Yazılım Mühendisi olarak çalışmaktayım. Boş zamanlarımda çoğunlukla açık kaynak projelerini destekliyorum.

Backend tarafında Java ve Kotlin programlama dillerinde iyiyim. Java programlama dilinde 7 yıldan, Kotlin Programlama Dilinde 1 yıldan fazla tecrübem var.

UI tarafında javascript kodluyorum. React Js ile 5+ proje ve Vue Js ile 1 proje tamamladım.

Çoğunlukla JVM dünyası için arka uç geliştiricisi olarak çalıştım, ancak bazı şirketler için de Fullstack Developer olarak çalıştım.

Kendimi takımda proaktif bir kişi olarak tanımlayabilirim. Demek istediğim, bir projeyi sıfırdan alabilirim ve onu bitirene kadar ilerleyebilirim. Ayrıca projeleri ve ekipleri yönetmek için bazı deneyimlerim oldu.

Son 4 yıldır çoğunlukla API’ler yazdım ve şirketler için 3. parti API Entegrasyonları yaptım.

Sosyal platformlarda hakkımda daha fazla bilgi edinebilirsiniz. Zamanım olduğunda Yazılım Tasarımı, Yazılım Mimarisi ve Programlama Dilleri hakkında yeni makaleler yazmayı seviyorum.
